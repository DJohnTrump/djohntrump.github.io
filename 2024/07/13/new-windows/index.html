Windows 11 全新开机设置流程
2024-07-13

1. 跳过开机联网激活

  1. 首先进入需要联网的界面，按下键盘上的 Shift + F10；
  2. 按下后会打开命令提示符，在其中输入 oobe\bypassnro 后回车；
  3. 使用上述命令后会重新启动，再进入联网界面后可以发现，右下角多了一个「我没有 Internet 连接」；
  4. 点击它之后进入下图界面，再点击「继续执行受限设置」，就可以跳过联网激活，进行后续的设置。

2. 卸载自带软件

3. 设置鼠标滑轮为自然方向

参考文章：Windows系统下更改鼠标的滚动方向

4. 修改系统字体为新微软雅黑

5. Windows 家庭版改为专业版

参考文章：win11专业版激活密钥（亲测有效）2个

省流：

  1. 断开网络后到系统输入密匙：J8WVF-9X3GM-4WVYC-VDHQG-42CXT，更新完成后联网；
  2. 打开终端， 输入 irm massgrave.dev/get.ps1 | iex 后回车；
  3. 弹出窗口选[1]；
  4. 回到系统查看会显示「激活成功」。

6. 系统版本升级到 23H2

解决无法获取 23H2 更新：KB5027397 Enablement Package for Windows 11 version 23H2 Feature Update

7. Microsoft Store 安装 PowerShell 7

8. 安装 WSL 和 Debian

参考文章: 如何使用 WSL 在 Windows 上安装 Linux

省流：使用管理员打开终端，输入 wsl –install -d Debian 后回车，然后重启计算机。

9. 键盘重映射

Microsoft Store 安装 PowerToys，映射 Copilot 键为 Ctrl(Right)：Copilot = Win(Left) + Shift(Left) + F23
